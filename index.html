Pillow

  • Форк Python Imaging Library (1995 года)
  • Нативный модуль для Python
  • Версии Python: 2.7, 3.3 — 3.6
  • Разработка ведется 3-месячными циклами
  • Сайт: python-pillow.org

Возможности Pillow

  • Кодеки для 17 форматов + декодеры еще для 18
  • Есть все для «модификации пользовательского контента»
  • Режимы: RGB, RGBA, L, LA
  • Рисование на зачаточном уровне, нет Безье, режимов смешивания
